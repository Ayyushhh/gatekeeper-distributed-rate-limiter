# gatekeeper-distributed-rate-limiter
A standalone distributed rate limiter supporting Token Bucket and Sliding Window algorithms with persistent state and concurrent request safety.

## 📚 What You'll Learn

Building this project from scratch will help you gain hands-on experience with production-grade backend engineering concepts, including:

### 🧠 Algorithms
- Token Bucket rate limiting
- Sliding Window rate limiting

### ⚡ Concurrency & Synchronization
- Race conditions and their impact
- Optimistic Locking
- Pessimistic Locking
- Atomic operations
- Preventing double token consumption
- Thread-safe request handling

### 🗄️ Data Persistence
- Database schema design
- Managing time-based state efficiently

### ⏱️ Time & State Management
- High-precision time calculations
- Token refill logic
- Window expiration
- Handling clock drift and elapsed time

### 🚀 Performance Engineering
